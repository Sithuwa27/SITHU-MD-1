# **App Name**: SITHU MD

## Core Features:

- WhatsApp Pair Code Integration: Securely link your WhatsApp account to the bot via a 8-digit pair code, bypassing QR scans.
- Intelligent Song Search Tool: An AI-powered tool that analyzes search queries to identify songs from lyrics, melodies, or partial names and suggests the best match for download.
- Direct Media Downloader: A streamlined interface for downloading high-quality audio files directly from WhatsApp once requested through the bot.
- Real-time Bot Status: A dashboard UI showing whether the bot is online, connected, and active.
- Interactive Command Generator: Visual interface to generate and copy bot commands for common tasks like search, lyrics display, and metadata editing.
- Session Management: Option to disconnect or logout the WhatsApp session directly from the web interface.

## Style Guidelines:

- Primary color: Electric Violet (#6179FF) symbolizing advanced technology and rhythmic energy.
- Background color: Deep Charcoal (#0D0E14), creating a high-contrast environment suitable for 24/7 dashboard use.
- Accent color: Celestial Blue (#00BAFF) to highlight interactive elements and connectivity states.
- Headline and body font: 'Space Grotesk', chosen for its machined, technical feel that aligns with bot management and digital media metadata.
- Linear, minimalist icons with subtle glow effects to denote music waves, wireless signals, and linking status.
- Compact, single-column dashboard layout on mobile and grid-based on desktop for rapid monitoring and input.
- Pulse animations on the 'Active' connection indicator and smooth height transitions when pairing codes are generated.
